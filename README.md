# Unitree Mujoco G1 Robot Documentation

This repository contains the documentation for using the Unitree Mujoco package with the G1 robot.

## GitHub Pages

The documentation is hosted on GitHub Pages at the following URL:
[https://KatzAIM.github.io/unitree_mujoco_g1_docs/](https://KatzAIM.github.io/unitree_mujoco_g1_docs/)

## About the Project

This project provides a comprehensive guide to the G1 robot simulation in Mujoco, including:
- Project structure and setup
- Joint information and configurations
- Running simulations using Python
- C++ examples for low-level control

For more information, please refer to the documentation website.
